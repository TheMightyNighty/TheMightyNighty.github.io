# Screenshots

Eigene Screenshots für die Projektkarten.

**Dateiname = Repo-Name**, z.B.:

- `jsonforms-designer.png`
- `ImpExpNL.jpg`

Unterstützte Formate: `.png`, `.jpg`, `.jpeg`, `.webp`

Wenn eine Datei hier liegt, wird sie statt des automatischen GitHub-Previews verwendet.
Nach dem Upload den Workflow **"Update Projects Data"** manuell auslösen
(GitHub → Actions → Update Projects Data → Run workflow) — dann wird `projects.json` aktualisiert.
